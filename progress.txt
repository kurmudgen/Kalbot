HOW THE KALSHI BOT WORKS (bot/dual_strategy.py)
================================================================

Every 5 minutes, runs this pipeline:

1. KILL SWITCH CHECK — S:\kalbot\STOP file exists? Skip cycle.

2. REGIME DETECTION (bot/regime_detector.py)
   - Downloads S&P 500 prices via yfinance
   - Classifies BULL/BEAR/VOLATILE based on 20-day trend + volatility
   - Currently: BEAR (-5.7% trend, 12% vol)
   - Adjusts Kelly: bull=1.3x, bear=0.6x, volatile=0.5x

3. SEC EDGAR MONITOR (bot/edgar_monitor.py)
   - Fetches 8-K filings, NLP classifies bullish/bearish
   - Telegram alerts for material events

4. RESOLUTION TRACKER (bot/resolution_tracker.py)
   - Checks if traded markets resolved → actual P&L
   - Feeds outcomes to trade memory + adaptive weights + Telegram

5. CROSS-MARKET CORRELATION (bot/cross_market.py)
   - Flags when CPI/Fed/GDP markets diverge unexpectedly

6. EARLY EXIT (bot/early_exit.py)
   - Take profit at 60%, stop loss at 30% (polling-based)

7. POLYMARKET SIGNAL — Crawls PM for sharp moves, signals only

8. MARKET MAKER — Spread orders on thin weather markets

9. DATA SNIPER — Pre-researches upcoming CPI/Fed/jobs releases

10. QUANTITATIVE STRATEGIES (6 modules):
    SP500: VIX-implied brackets with VRP haircut
    BITCOIN: Jump-diffusion Monte Carlo (10K paths)
    ECON: Cleveland Fed + GDPNow + FedWatch (very conservative, >12% edge required)
    TREASURY: Yield change distribution
    GAS: EIA weekly + crude correlation
    FOREX: Implied vol bracket pricing

11. MARKET SCANNER — Raw Kalshi REST API by series ticker
    ~573 target markets, classifies by prefix, rejects sports bundles

12. LOCAL FILTER — qwen2.5:7b via Ollama
    Pre-filters: skip no-volume, resolved, recently scored
    Caps at 200/cycle, returns cached passes too

13. ENSEMBLE ANALYST — 3-model pipeline:
    Perplexity (researcher, web search) → Claude + DeepSeek (independent evaluators)
    Consensus: 2/3 majority + confidence floor >0.65
    Calibration: Platt scaling → edge dampening → spread penalty
    Adaptive weights: inverse-Brier per model per category

14. EXECUTOR — Safety gauntlet before every trade:
    Kill switch → bracket dedup → confidence floor → orderbook depth →
    seasonal adjustment → historical bias → regime Kelly → correlation guard →
    category thresholds → spend caps → paper/live mode
    Position sizing: Quarter-Kelly * regime_multiplier

15. STRATEGY REBALANCER — Shifts budget toward winners
    Default: WEATHER 30%, SP500 20%, BTC 12%, SNIPER 8%, ECON 8%,
    TREASURY 8%, EMERGING 6%, GAS 5%, FOREX 5%

16. AUTO-SCALER — $50→$200→$500→$1000 as win rate proves out
