=== KEY FINDINGS ===

1. Weather is our strongest category (79-90% WR, 0.07-0.19 Brier)
2. TSA is weakest (56-64% WR) — needs domain-specific prompt or TSA ground truth integration
3. Model generalizes well (train-val gap < 5%)
4. High-gap accuracy on validation only 36% — model doesn't reliably beat market when it disagrees
5. v1 prompt outperforms v2/v3 — simple > structured for this model
6. Average confidence too low (0.41) — model is too uncertain, which is actually conservative/safe
